# Travel-Fly - Homework Four

## Project Overview

Travel-Fly is a responsive travel booking website designed as part of a web development course assignment. The website showcases a modern travel agency platform with promotional tour packages and an integrated booking form.

## Features

- **Fixed Navigation Header** - Easy access to navigation links (Home, About, Tours, Special Offers, Blog, Contacts)
- **Hero Section** - Eye-catching landing section with call-to-action button
- **Tour Promotions** - Featured destination packages (Israel, USA, Australia) with pricing and "Learn More" buttons
- **Booking Form** - Interactive form for users to submit travel booking requests including:
  - Personal information (name, email)
  - Travel details (country, hotel)
  - Check-in and check-out dates
  - Guest count (adults, children, rooms)
  - Special message field
- **Responsive Design** - Built with SCSS for maintainable and scalable styling
- **Footer** - Copyright information and social media links

## Technologies Used

- HTML5
- SCSS/CSS
- Responsive Web Design
